## Running the application

In one terminal window:

```bash
yarn install
yarn start
```

In another terminal window:

```bash
pip install -r ./requirements.txt
uvicorn main:app --app-dir app --reload
```

## Running tests

React:

```bash
yarn install
yarn test
```

Then press `a` to run all tests.

Python:

```bash
pip install -r ./requirements.txt
PYTHON_PATH=app pytest --verbose
```
